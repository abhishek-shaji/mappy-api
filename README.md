# Mappy API

## Description

[Nest.js](https://github.com/nestjs/nest) Backend for [Mappy.Earth](https://apps.apple.com/us/app/mappy-earth/id1590029147) App

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
